# 🎮 Mini Games Collection

> 精选好玩的HTML5小游戏合集

## 🕹️ 游戏列表

### 1. 文字冒险 - 今日的选项
- 类型：剧情向文字冒险
- 玩法：选择不同的选项，影响故事走向
- 特色：好感度系统、金币收集、多结局

📁 路径：`text-adventure/index.html`

### 2. AI恋人 - 小狼狗
- 类型：AI聊天陪伴
- 玩法：和AI角色聊天互动
- 特色：智能回复、约会系统、送礼互动

📁 路径：`ai-companion/index.html`

### 3. 脑洞大开 - 百万富翁
- 类型：知识问答
- 玩法：答对12题平分100万
- 特色：脑洞问题、AI相关问答、锦囊道具

📁 路径：`brain-hole/index.html`

### 4. AI打砖块
- 类型：经典街机
- 玩法：经典打砖块玩法
- 特色：AI辅助模式、关卡升级、触摸控制

📁 路径：`breakout/index.html`

---

## 🚀 如何运行

直接在浏览器中打开 `index.html` 文件即可游玩！

```bash
# 使用Python启动本地服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```
